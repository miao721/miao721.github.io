一、申请所需要提供的资料

您需送出表单后才能进入管理员审核后台，为了保证您确实有需要申请h岛匿名版，您需要填写详细以下信息:

  • 名称
  • 联络讯息
  • 介绍

*联络讯息是必须填写的，但并不需要完全真实，您可以随机填写一个字符串，一般情况下我们没有联系您的需要，在以上前提下，您可以填写不真实的讯息，但是若您认为您与官方可能有必要进行沟通，请填写真实的email信息，若有必要，我们会联络您。

二、完全的匿名性

申请成功后您可不必成为该小岛的版主即可正常发言(如果您有饼干)，h岛遵守网友匿名的权利，无论是前台还是后台，都将以完全匿名的形态呈现用户的信息，若您需要成为版主，则需要您提供您的腾讯QQ账号联络我们的官方客服为您开启权限。

三、小岛管理者的权限

若您为第一位申请版主，则我们默认为您开启版主/总岛主的权限。

【总岛主】您能够拥有的权限:

  • 1、编辑您的小岛的信息
  • 2、关闭您的小岛(关闭后无法自行打开)
  • 3、任命版主(如果该用户存在在我们的数据库中)
  • 4、版主拥有的权限

【岛主】您能够拥有的权限

  • 1、封id（饼干）
  • 2、管理您版块的帖子

四、何种情况下h岛会关闭您的小岛

1.版块一个月内少于10个主题帖

2.版块三十天内无人发布内容

3.版块包含了我们不支持的内容，如：盗用软件、黑客、破解、滥发、攻击网络服务器、种族歧视、反人道主义、不适当的服务器活动以及其他违法的行为

另外我们不会通过主题相同的版块，谢谢。
